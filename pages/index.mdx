# Final Project | Open Recruitment Lab RPL 2023

**Final Project Open Recruitment Lab RPL 2023** adalah upaya kolaboratif yang melibatkan lima kelompok dari 30 peserta, dengan enam anggota ( _3 Frontend & 3 Backend_ ) dipilih oleh admin untuk setiap kelompok. Proyek ini bertujuan untuk memberikan platform bagi peserta untuk menampilkan kemampuan mereka dalam bekerja sama dan mengembangkan sebuah aplikasi untuk masalah dunia nyata.
