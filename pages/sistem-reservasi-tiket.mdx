# Sistem Reservasi Tiket 

Membuat sistem reservasi tiket nonton dan tempatnya untuk Bioskop Drive-In pada suatu parkiran outdoor sebuah mall. Sistem ini dibuat untuk mempermudah para pengunjung untuk memesan tiket film kesukaannya dengan tempat parkir mobil yang diinginkan.

User bisa melihat film apa saja yang sedang tayang dengan jam tayangnya masing-masing. Kemudian user bisa memilih untuk melakukan pemesanan tiket dengan memilih posisi parkiran yang diinginkan. Setelah memilih, user dapat menyelesaikan proses pemesanan dengan mengisi data diri dan pembayaran. Setelah itu, user akan mendapatkan tiket yang berisi kode reservasi dan posisi parkir yang dipesan. User juga bisa melihat riwayat pemesanan tiketnya.

Selain user, sistem ini juga memiliki fitur untuk admin. Admin bisa melihat daftar film yang sedang tayang, menambahkan film baru, mengubah data film, dan menghapus film. Admin juga bisa melihat daftar tiket yang sudah dipesan.
