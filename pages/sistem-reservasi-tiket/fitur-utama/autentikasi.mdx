# Autentifikasi

User dapat melakukan Register dan Login menggunakan kredensial sewajarnya sebuah sistem reservasi tiket perlukan.

Terdapat dua role yaitu Admin dan User biasa dimana akun Admin dibuat melalui orang dalam, jadi tidak harus memiliki halaman khusus untuk daftar akun Admin.

Tentu saja dengan adanya role Admin, maka User biasa tidak dapat melakukan CRUD pada data yang berhubungan dengan Admin.
