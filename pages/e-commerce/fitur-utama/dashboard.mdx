# Dashboard  [ADMIN]

### Dashboard User (pembeli)
Terdapat halaman dashboard user, di mana user dapat :

1. Dapat melihat dashboard
    Dashboard dapat menampilkan produk.
2. Dapat melihat produk
    User dapat melihat dan memilih produk.

### Dashboard Admin (penjual)
Terdapat halaman dashboard admin/penjual, di mana admin dapat :

1. Dapat melihat dashboard
    Dashboard dapat menampilkan daftar produk yang dijual oleh admin.
2. Dapat CRUD produk
    User dapat menambahkan, melihat dan memilih produknya, mengupdate produk, dan menghapus produk.
